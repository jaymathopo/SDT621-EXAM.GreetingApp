# GreetingApp

A simple Windows Forms application that asks the user to enter their name and displays a greeting message.

## How to run

1. Open the folder in Visual Studio 2022.
2. Open `GreetingApp.sln` or `GreetingApp.csproj`.
3. Press **F5** or click **Start**.

## Requirements

- Windows
- Visual Studio 2022
- .NET 8 SDK with Windows Forms support
